使用Github Pages搭建Hexo个人博客的方法及注意事项

Created2024-06-29|Updated2024-10-27|Code

准备工作

  • 下载安装 node.js （官网下载安装）
  • 下载安装 git （官网下载安装）

本地搭建 hexo 静态博客

  • 安装 hexo 框架: 打开 cmd 运行

$ npm install -g hexo-cli

  • 新建一个文件夹，如 MyBlog ，进入该文件夹内，右击运行 git ，输入：
    $ hexo init
  • 生成完 hexo 模板，安装 npm ，运行：

$ npm install
没错，博客的主体部分到此已经完成了，来看看效果吧。运行：
$ hexo server

这时候打开浏览器，输入 localhost:4000 就可以看到博客目前的样子了。然后按 Ctrl + C 就可以继续下面的操作了。

将博客与GitHub关联

  • 注册 GitHub 账号：如果还没有 GitHub 账号的，需要先注册一个

  • 在 Github 上创建名字为 XXX.github.io 的项目，XXX为自己的 GitHub 用户名。

  • 打开本地的 MyBlog 文件夹项目内的 _config.yml 配置文件，将其中的 type 设置为 git :

  • 运行：
    npm install hexo-deployer-git --save

  • 本地生成静态文件，并将静态文件推送至GitHub，运行：
    hexo g
    hexo d
    此时，打开浏览器，访问 http://your-name.github.io ，到此为止你的博客已经建设完毕了。
